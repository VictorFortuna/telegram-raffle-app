
Проект: Telegram Stars
Концепция: Игра для розыгрыша Telegram Stars. Пользователи делают ставки (1 звезда), победитель получает 70%, организатор 30%. Розыгрыш запускается автоматически при достижении нужного количества участников.

Функции для пользователей:
Автоматическая регистрация через Telegram
Проверка баланса звезд через API
Ставка в 1 звезду (при недостатке - предложение купить)
Участие в розыгрыше после ставки
Получение результатов по завершении

Функции для администратора:
Настройка параметров игры (количество участников, размер ставки, процент распределения)
Отмена розыгрыша
Просмотр статистики (пользователи, участники, звезды, история)

Технические требования:
Интеграция с Telegram API
База данных (users, bets, raffles)
Хостинг на Railway или аналогах
Безопасность данных

Этапы работы:
Регистрация и сохранение данных
Ставка и пополнение копилки
Розыгрыш и распределение звезд
Уведомления о результатах
Администрирование

Цели:
Удобное приложение для розыгрышей
Привлечение пользователей
Гибкость управления

Особенности:
